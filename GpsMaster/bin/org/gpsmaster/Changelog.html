New features in this version:
  • Chart: two new x-axes (time, duration) with selector
  • Wikipedia & photo waypoints properly loaded from GPX file again
  • GpsMaster specific extensions now in "gpsm" namespace (gpsm:color, gpsm:activity)

Bugfixes:
  • Deletion of track/segment no longer doubles duration

known issues:
  • Chart design needs improvement
  • some GPX file extensions will not be written to file when saving
  • OSM download / list of relations: query results sometimes rather erratic
  • metric units still hardcoded in some parts of the code
