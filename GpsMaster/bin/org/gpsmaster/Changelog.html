New features in this version:
  • support for IGC format (written by glider flight recorders)
  • Scalebar on map (on/off in config)
  • Unit system NAUTICAL fully supported (set in config)
  • Numerical values from waypoint extensions now selectable for chart Y-Axis

Bugfixes/internals:
  • path finders (MapQuest, YOURS) working again
  • Unit system rewritten completely
  • GPX Tree code consolidated
  • New CornerLayoutManager
  • No more hardcoded units

known issues:
  • Chart axes selector design needs improvement
  • some GPX file extensions will not be written to file when saving
  • OSM download / list of relations: query results sometimes rather erratic
