New features in this version:
  • basic database support (store GPS files in relational database)
  • loader for NMEA files
  • loader for CPO files
  • Measure distance: direct line between measure points
  • http proxy support
  • icons for dialog windows
  • email address for donations via paypal: donation@gpsmaster.org

Bugfixes/internals:
  • chart code no longer active when map is not showing
  • elevation widget no longer moving to the right on consecutive invocations
  • scan gpx extensions when map is enabled
  • gross rise/gross fall values multiplied on update
  • map painter code rewritten to allow more flexibility
  • improved calculation of directional arrows angle
  • minor improvements & bugfixes

known issues:
  • Chart axes selector design needs improvement
  • some GPX file extensions will not be written to file when saving
  • OSM download / list of relations: query results sometimes rather erratic
